## 📁 Featured Projects

### 🔵 [Micro-Scale Displacement Measurement — Computer Vision](https://github.com/Nikhilvinayagamurthy/Measurement-of-Micro-Scale-Displacement-Using-Computer-Vision)
OpenCV morphology pipeline for automated crack detection and displacement quantification in metal-forming specimens. Validated across 14 samples — sub-pixel accuracy at ≤10 µm resolution. Published in IEEE.

### 🔵 [Reverse Engineering → Parametric CAD](https://github.com/Nikhilvinayagamurthy/Reverse-engineering-parametric-CAD)
Semi-automated pipeline using RANSAC + NX Open API to convert physical parts into parametric CAD models. Reduced manual modelling time by 60–70%.

### 🔵 [Sustainable Heated Bed System for FDM 3D Printing](https://github.com/Nikhilvinayagamurthy/Design-and-development-of-a-sustainable-heated-bed-system-for-3D-printing)
Energy-efficient heated bed design with embedded C++ control logic. Achieved 30% reduction in support material and improved thermal uniformity.

### 🔵 [Rover Tire — Additive Manufacturing Optimisation](https://github.com/Nikhilvinayagamurthy/Additive-manufacturing-rover-tire)
Dual-material FDM tyre (TPU 95A + PETG) optimised for an autonomous stair-climbing rover. Structural performance validated through real-world testing on Prusa XL.

### 🔵 [Autonomous Stair-Climbing Rover — Design & Virtual Testing](https://github.com/Nikhilvinayagamurthy/Autonomous-stair-climbing-rover-Design-modification-virtual-testing)
Design modification and virtual testing of a stair-climbing rover at TU Clausthal's Industry 4.0 Lab. Full simulation-to-prototype workflow.
